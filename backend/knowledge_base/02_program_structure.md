# MathPath Program Structure

MathPath uses a structured learning journey divided into age-appropriate and level-appropriate programs. The program supports children from the early learner stage to advanced middle-school readiness.

## Overall Learning Journey

The visible program structure includes:

| Program / Stage | Suggested Age Group | Duration | Main Focus |
|---|---:|---:|---|
| Young Learner Program | 5–7 years | 10 months | Early number sense, abacus basics, visualisation, foundational operations |
| Preparatory Level | 8 years onwards | 4 levels, 4 months each | Multiplication, division, decimals, BODMAS, word problems, mental picture memory |
| Intermediate Level | Advanced learners / middle-school preparation | 4 levels, 4 months each | Complex operations, decimals, integers, squares, strategic thinking |
| Master Module | Advanced module | Compact advanced module | Percentages, roots, cubes, profit and loss, simple interest, decimal operations |
| Bridge Course | Late joiners, especially older children | Fast-track foundational route | Helps late joiners enter the MathPath journey without unnecessary delay |

## Program Philosophy

Every child does not enter MathPath at the same age or skill level. Therefore, MathPath uses different programs to support different learner needs.

- Younger children need concrete number understanding, playful visual learning, and gradual skill building.
- Children from age 8 onwards need stronger arithmetic fluency, multiplication, division, decimals, and school application.
- Older children need fast-track support, advanced operations, and connection with school concepts.
- Advanced learners need higher-level applications like percentage, roots, profit and loss, and interest.

## Recommendation Logic for Bot

When the parent provides the child’s age/class, the bot can recommend:

- Age 5–7: Young Learner Program
- Age 8 or above with basic foundation needs: Preparatory Level
- Older child with prior abacus or strong arithmetic base: Intermediate Level
- Class 5–7 late joiner: Bridge Course followed by suitable level placement
- Advanced child preparing for complex arithmetic and school application: Master Module

The bot should never make a final level-placement guarantee. It should say that final placement will be confirmed by the MathPath team after understanding the child’s current ability.

## Suggested Bot Answer

MathPath has a complete learning journey starting from the Young Learner Program for ages 5–7, then Preparatory Levels for children 8 years onwards, followed by Intermediate Levels and a compact Master Module. For older children who join late, MathPath also offers a Bridge Course so they can catch up without unnecessary repetition.

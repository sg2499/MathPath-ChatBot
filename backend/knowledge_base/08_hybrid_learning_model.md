# MathPath Hybrid Learning Model

MathPath follows a hybrid learning model that combines structured class learning with regular app-based home practice.

## Main Elements

The model includes:

- Weekly structured class learning
- Daily Practice Sheets
- App-based practice support
- Auto-checking
- Smart tracking
- Gamified practice
- Parent feedback
- Mock tests
- Level assessments
- Report cards
- Missed-practice alerts

## Why Hybrid Learning Works

Children learn best when classroom instruction is supported by regular, short, and meaningful practice. MathPath’s hybrid model helps children practice consistently without making the process stressful.

The class introduces concepts, the app supports regular practice, and tracking helps parents understand whether the child is practicing properly.

## Daily Practice Sheets

Daily Practice Sheets help children build habit and fluency. Instead of waiting for one weekly class, children continue learning in small daily steps.

## Auto-Checking and Feedback

Auto-checking helps reduce delay in correction. Children and parents can identify mistakes faster, which improves timely progress.

## Gamified Learning

Gamified learning makes daily home practice more exciting and suitable for modern learners. It encourages children to stay regular and motivated.

## Parent Feedback and Tracking

Parents get better visibility into their child’s progress through tracking, reports, and alerts. This reduces uncertainty and helps parents stay involved.

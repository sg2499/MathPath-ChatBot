# MathPath FAQ and Program Details - Latest Internal Update

This file adds the latest official FAQ-style information for the MathPath chatbot. It should be used along with the existing MathPath knowledge base. Do not delete or replace older knowledge-base files unless specifically instructed.

## Q1. What is Math Path Abacus?

Math Path Abacus helps children fall in love with maths by making it simple, visual, and fun.

The program is designed for children from UKG to Class 8 and combines:

- Abacus mastery
- Modern cognitive training
- Visualisation techniques
- School syllabus integration

Core positioning:

Unlock your child's true maths potential with a unique hybrid learning experience that blends abacus mastery, powerful visualisation techniques, and seamless school syllabus integration.

## Q2. How does MathPath cater to different age groups?

For enrolment/program-overview questions, the chatbot must show only these three entry programs. Intermediate Level and Master Module are later progression stages for enrolled students and should not be listed as separate entry programs unless the user specifically asks about full progression or advanced levels.

MathPath has different entry programs for different age groups and class levels.

### Young Learner Program

- Suitable for: UKG, Class 1, and Class 2
- Duration: 10 months
- Focus: Foundational operations in abacus
- Progression: After completion, the child is promoted to Preparatory Level 2

### Preparatory Level 1

- Suitable for: Class 3 and Class 4
- Duration: 4 months
- Focus: Visual and operational fluency
- Progression: After completion, the child is promoted to Preparatory Level 2

### Bridge Course

- Suitable for: Class 5 to Class 8 late joiners
- Duration: 10 to 12 months
- Structure: 4 levels are clubbed into one compact program
- Progression: After completion, the child is promoted to Intermediate Level 1

## Q3. What is the total program duration from start to end?

The total duration is approximately 4 years, depending on the child's entry level.

## Q4. Will MathPath improve children's maths?

MathPath aims to improve children's maths ability by working on:

- Concentration
- Whole-brain development
- Visual memory
- Picture memory
- Numerical thinking
- Mental calculation confidence

MathPath uses numbers as a medium of training and communication. Each class also includes a 30-minute Brain Spark Happy Session where children learn maths in a fun way.

## Q5. What is the class frequency and duration?

MathPath provides:

- One class per week
- Each class duration: 2 hours
- Monthly classes: 4 classes
- Daily practice through the automated student portal
- Daily Practice Sheet submission through the portal

## Q7. What is the batch size for each class?

Each batch allows a maximum of 10 to 12 students so that children receive proper one-to-one attention and the teacher can distribute time properly.

## Q8. Do you have any academic session for new admission?

There is no fixed academic session for admission.

Admissions happen throughout the year. A child's learning starts from the day they join the class. Assessments happen based on individual level completion, not group completion.

## Q9. Are there any competitions?

Yes. MathPath conducts annual competitions.

Students from Level 2 onward are eligible to participate and compete. This is a full-day event where results are declared and prizes are distributed on the same day.

## Q10. What class-day options are available?

Abacus classes are mainly after-school activities. MathPath offers:

- Weekday evening classes from 5 PM onward
- Saturday batches
- Sunday batches
- Morning, afternoon, and evening batch options on weekends

Exact batch availability should be confirmed with the MathPath team.

## Q11. What is the assessment and certification process?

Every level has a level-end assessment.

- Minimum score required for promotion: 75%
- After successful assessment, a certificate is issued for every level
- Promotion happens based on the child's individual level completion

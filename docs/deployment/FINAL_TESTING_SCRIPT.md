# Final Testing Script

Use these questions after deployment to validate the chatbot.

## Program recommendation

- My child is 6 years old. Which program is suitable?
- My child is in Class 3. Can MathPath help?
- My child is in Class 6 and has never done abacus. Is it too late?

## Parent concerns

- My child is weak in maths and afraid of calculations.
- Will this help with school maths?
- How much daily practice is required?
- How do parents know if the child is practicing?

## MathPath-specific features

- What is the Bridge Course?
- How does the hybrid learning model work?
- What are Daily Practice Sheets?
- What are mock tests and report cards?

## Lead capture

- I want to book a demo.
- Please call me for admission.
- What are the fees?

Expected result: the lead form should appear or the bot should guide the parent to contact MathPath.
